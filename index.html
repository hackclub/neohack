Requirements

Using Hackatime

Track the time you've spent on your plugin using Hackatime.

5 Hours Minimum

You must spend atleast 5 hours on your Neovim/Vim plugin. Some good ways to spend this time is:

  • Writing proper documentation (use jekyll, or even github pages)
  • Writing unit tests
  • Adding more features!

Proper Documentation

Include a comprehensive README.md with install instructions for the most popular package managers (ie packer, lazy.nvim, etc)
